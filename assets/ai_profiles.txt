# AI Profiles - Each profile defines an AI personality
# Values are hot-reloaded every 10 seconds when changed
#
# Parameters:
#   position_tolerance - How close AI needs to be to target before stopping (pixels)
#   shoot_range - Distance from basket at which AI will shoot (pixels)
#   charge_min - Minimum charge time before releasing shot (seconds)
#   charge_max - Maximum charge time for shots (seconds)
#   steal_range - Distance at which AI will attempt steal (pixels)
#   defense_offset - How far from own basket AI defends (pixels)
#   min_shot_quality - Minimum shot quality (0.0-1.0) before AI will shoot
#                      Based on position heatmap: 0.0=anywhere, 1.0=perfect only
#   pressure_distance - How close AI stays to ball carrier (lower = more pressure)
#   aggression - How aggressively AI pursues (0.0=passive, 1.0=relentless)
#   defensive_iq - How well AI positions on shot line (0.0-1.0)
#   steal_reaction_time - Seconds before AI attempts steal after entering range
#                         Lower = faster reflexes (0.10-0.30, human average ~0.20)
#   button_presses_per_sec - Max button presses per second (human range: 6-15)
#                            Limits how fast AI can mash pickup/steal
#   position_patience - Multiplier for seek utility (0.3-2.0)
#                       Higher = more willing to seek better positions
#   seek_threshold - Minimum utility to seek better position (0.05-0.20)
#                    Higher = shoots more quickly from current position
