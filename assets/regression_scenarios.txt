# Regression Test Scenarios
# =========================
#
# Each scenario defines a specific game configuration for visual regression testing.
# The regression script runs each scenario, captures a screenshot, and compares to baseline.
#
# Format:
#   scenario: <name>           Unique identifier for this test case
#   level: <level_name>        Level to load (by name, not index)
#   palette: <palette_index>   Color palette to use (0-indexed)
#   viewport: <width> <height> Window size in pixels
#   wait_frames: <frames>      Frames to wait before capturing screenshot
#   description: <text>        Human-readable description of what's being tested
#
# Notes:
#   - Scenario names are used for baseline file names (e.g., game_start.png)
#   - Comments start with # and are ignored
#   - Blank lines are ignored
#   - All fields are required for each scenario
