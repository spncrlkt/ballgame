## per_level_summary
SQL:
```
SELECT m.level_name, COUNT(DISTINCT m.id) AS matches, AVG(m.score_left + m.score_right) AS goals_per_match, SUM(ps.shots_attempted) AS shots_attempted, SUM(ps.shots_made) AS shots_made, AVG(CASE WHEN m.score_left + m.score_right = 0 THEN 1.0 ELSE 0.0 END) AS scoreless_rate, AVG(ps.avg_shot_quality) AS avg_shot_quality FROM matches m JOIN player_stats ps ON ps.match_id = m.id GROUP BY m.level_name ORDER BY scoreless_rate DESC, shots_attempted ASC
```
